💬 Tutorial: Ask Mode

Aprende a usar Ask Mode de Copilot para hacer preguntas efectivas sobre código

💬

¿Qué es Ask Mode?

Ask Mode es tu herramienta perfecta para hacer preguntas sobre código y conceptos de programación sin modificar nada. Es como tener un experto programador siempre disponible.

🎯 ¿Cuándo usar Ask Mode?

🧠Entender Código

Cuando necesitas que te expliquen qué hace una función, clase o algoritmo específico

📚Aprender Conceptos

Para entender patrones de diseño, mejores prácticas o nuevas tecnologías

🔍Debugging Mental

Para analizar problemas potenciales sin hacer cambios en el código

⚡Consultas Rápidas

Para respuestas inmediatas que te mantengan en tu flujo de trabajo

🚀 ¿Listo para empezar?

En los próximos pasos aprenderás a acceder y usar Ask Mode de manera efectiva.

Cómo Acceder a Ask Mode

Hay varias formas de abrir Ask Mode. Aprende los métodos más rápidos para no interrumpir tu flujo.

🎹 Método Principal

Ctrl + Alt + I

Abre el Chat view, luego selecciona "Ask" del dropdown

📱 Métodos Alternativos

MENÚ

Copilot Menu

Haz clic en "Open Chat" en la barra de título de VS Code

QUICK CHAT

Chat Rápido

Ctrl+Shift+Alt+L
Para preguntas rápidas sin sesión completa

CONTEXTUAL

Click Derecho

Selecciona código → Click derecho → Menú Copilot

Tus Primeras Preguntas

Aprende a formular preguntas efectivas y qué esperar en las respuestas de Ask Mode.

🎯 Ejemplo Básico

¿Qué hace esta función?
Esta función parece ser un método de validación que verifica si un email tiene el formato correcto. Utiliza una expresión regular para comprobar que:
• Tiene caracteres válidos antes del @
• Tiene un dominio después del @
• Termina con una extensión válida

Retorna true si el email es válido, false en caso contrario.

🔄 Tipos de Respuestas

📝Explicaciones

Texto descriptivo sobre cómo funciona el código o concepto

💻Bloques de Código

Ejemplos de código que puedes copiar manualmente

🔗Enlaces

Referencias a documentación o recursos útiles

📋Listas

Pasos ordenados o puntos clave organizados

Añadir Contexto Efectivo

El contexto es clave para obtener respuestas precisas. Aprende a proporcionar la información correcta.

🎪 Formas de Añadir Contexto

SELECCIÓN

Código Seleccionado

Selecciona código antes de preguntar para contexto automático

# VARIABLES

Menciones #

#archivo.js, #codebase, #function

ADD CONTEXT

Botón Contexto

Usa "Add Context" para adjuntar archivos específicos

🎬 Ejemplo con Contexto

En #UserService.js, ¿por qué esta función puede fallar con arrays vacíos?
Mirando el archivo UserService.js, veo que la función getUserById() no valida si el array de usuarios está vacío antes de hacer la búsqueda. Esto puede causar:

1. Error de referencia: Si users.length es 0, el método find() retornará undefined
2. Comportamiento inesperado: El código asume que siempre habrá usuarios

Solución recomendada: Agregar validación al inicio de la función.

🏆 Variables de Contexto Útiles

  • #codebase - Todo tu proyecto
  • #file.js - Archivo específico
  • #function - Función específica
  • #selection - Código seleccionado
  • #problems - Errores del workspace
  • #terminal - Output del terminal

Mejores Prácticas

Consejos avanzados para sacar el máximo provecho de Ask Mode y obtener respuestas de calidad.

🏆 Reglas de Oro

🎯Sé Específico

En lugar de "¿está bien este código?", pregunta "¿este código maneja correctamente los errores de red?"

📋Da Contexto

Menciona el framework, versión, o problema específico que intentas resolver

🔄Itera

Haz preguntas de seguimiento para clarificar o profundizar en las respuestas

📝Selecciona Código

Siempre selecciona el código relevante antes de hacer la pregunta

⚡ Referencia Rápida de Comandos

Abrir Chat Ctrl+Alt+I
Quick Chat Ctrl+Shift+Alt+L
Siguiente bloque Ctrl+Alt+PageDown
Bloque anterior Ctrl+Alt+PageUp

❌ Evita Estos Errores

  • Preguntas muy vagas como "¿qué hago?"
  • No proporcionar contexto sobre tu objetivo
  • Esperar que Ask Mode modifique tu código
  • No especificar el lenguaje o framework
  • Hacer muchas preguntas en una sola consulta
