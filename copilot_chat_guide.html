¿Qué es Copilot Chat?

💬 Chat Inteligente

Conversaciones naturales con IA sobre tu código

🔍 Análisis de Código

Explica, optimiza y debug tu código

🚀 Generación Rápida

Crea funciones, tests y documentación

Entiende tu proyecto y archivos abiertos

Requisitos Previos

1

Visual Studio Code

Versión actualizada de VS Code

2

Suscripción GitHub Copilot

Cuenta activa de GitHub Copilot (individual o business)

3

Extensión GitHub Copilot

Instalar desde el marketplace de VS Code

Instalación

1

Abrir VS Code

Inicia tu editor favorito

2

Ir a Extensiones

Presiona Ctrl+Shift+X (Windows/Linux) o Cmd+Shift+X (Mac)

3

Buscar "GitHub Copilot"

Instala tanto "GitHub Copilot" como "GitHub Copilot Chat"

4

Iniciar Sesión

Autoriza VS Code con tu cuenta de GitHub

Interfaz de Copilot Chat

🎯 Panel Lateral

Icono de chat en la barra lateral izquierda

💬 Chat Inline

Ctrl+I para chat dentro del editor

🔍 Quick Chat

Ctrl+Shift+I para chat rápido

📋 Menú Contextual

Click derecho en código seleccionado

Comandos Básicos

Atajos de Teclado:

Ctrl+I - Chat inline en el editor

Ctrl+Shift+I - Quick chat (ventana emergente)

Ctrl+Shift+P - Paleta de comandos → "Copilot"

Comandos de Chat:

/explain - Explica el código seleccionado

/fix - Arregla errores en el código

/tests - Genera tests unitarios

/doc - Genera documentación

Ejemplos Prácticos

Ejemplo 1: Explicar Código

Selecciona una función y pregunta:

"¿Qué hace esta función?"

"/explain esta función paso a paso"

Ejemplo 2: Generar Código

"Crea una función que ordene un array de objetos por fecha"

"Genera un componente React para un formulario de login"

Ejemplo 3: Debug

"/fix este error de JavaScript"

"¿Por qué mi API no está funcionando?"

Mejores Prácticas

  • Sé específico: Proporciona contexto claro sobre lo que necesitas
  • Usa el código seleccionado: Selecciona el código relevante antes de preguntar
  • Aprovecha los comandos: Usa /explain, /fix, /tests para tareas específicas
  • Itera: Haz preguntas de seguimiento para refinar las respuestas
  • Verifica: Siempre revisa y prueba el código generado
  • Contexto del proyecto: Copilot entiende tu workspace, úsalo
